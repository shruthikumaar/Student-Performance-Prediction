# Student Performance Prediction

## Project Overview

This Machine Learning project predicts student marks based on the number of study hours.

The model is trained using Linear Regression and can estimate expected marks for a given number of study hours.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Joblib

## Project Structure

## How to Run

### Install Dependencies

pip install -r requirements.txt

### Train the Model

python train.py

### Make Predictions

python predict.py
